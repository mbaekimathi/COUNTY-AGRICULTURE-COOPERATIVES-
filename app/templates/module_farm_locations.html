{{ i("spark") }} How to use this list

  1. Find someone quickly — Use the search box (name, membership or farmer code, phone, ward, county).
  2. Optional area filters — Expand Filter by admin area to narrow by county, sub-county, or ward. Leave every box unchecked to include all farmers.
  3. See map detail — Click a row for one farm’s GPS, estimated boundary corners, and preview map. Use the row checkboxes to compare several farms at once.

Estimated boundaries are square plots derived from recorded land size in acres, centred on each farmer’s GPS — useful for planning, not a legal survey. {% if session.get('employee_role') %} Open farmer management to fix coordinates or land size. {% endif %}
